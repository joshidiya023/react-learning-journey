# Day 1 - Building a Mini React

## What I Learned

Today, I explored how React works behind the scenes by implementing simplified versions of `React.createElement()` and `ReactDOM.render().

### Concepts Covered

* Creating HTML elements dynamically using `document.createElement()`
* Setting element properties such as:

  * `className`
  * `id`
  * `textContent`
* Applying inline styles using `Object.assign()`
* Building a custom `createElement()` function similar to React
* Creating a custom `React` object with a `createElement()` method
* Creating a custom `ReactDOM` object with a `render()` method
* Rendering elements inside a root container
* Understanding the difference between `append()` and React's rendering behavior
* Learning how React replaces content inside the root element during rendering

## Code Highlights

Implemented a simplified version of:

```javascript
React.createElement()
ReactDOM.render()
```
to understand how React creates and renders elements internally.
