## 🚀 What I Learned

## 💻 Concepts Practiced

### JSX

* Writing HTML-like syntax inside JavaScript.
* Babel converts JSX into `React.createElement()`.

### JSX Expressions

* Embedded JavaScript values inside JSX using curly braces `{}`.

### Inline Styling

* Applied CSS using JavaScript objects.

**Example:**

```javascript
const style = {
  backgroundColor: "orange",
  color: "white"
};
```

### Props

* Passed data from parent components to child components.
* Accessed data using `props.name` and `props.age`.

**Example:**

```jsx
<App name="Diya" age={22} />
```
